# 🔥 LLM Evaluation Platform

A modular LLM evaluation system that benchmarks multiple language models using datasets, automated scoring, and a Streamlit dashboard.

---

## 🚀 Features

- Compare multiple LLMs (GPT, LLaMA via Ollama)
- Automated evaluation using LLM-as-a-judge
- Latency tracking for performance benchmarking
- Model comparison dashboard (Streamlit)
- CSV-based experiment tracking

---

## 🧠 Architecture

- `llm_runner.py` → handles model inference
- `evaluator.py` → LLM-based evaluation
- `main.py` → runs experiments
- `app.py` → dashboard UI
